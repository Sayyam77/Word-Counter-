"# Word Counter Studio

A modern, feature-rich text analysis and manipulation tool that provides comprehensive writing statistics and text processing capabilities.

## Features

### Text Analysis
- **Word Count**: Accurate word counting with support for hyphenated and apostrophe words
- **Character Count**: Total characters and characters without spaces
- **Sentence & Paragraph Count**: Automatic sentence and paragraph detection
- **Average Word Length**: Calculate average word length in your text
- **Reading Time**: Estimate reading time based on standard reading speed
- **Speaking Time**: Estimate speaking duration for your text

### Advanced Statistics
- **Unique Words**: Track the number of distinct words
- **Lexical Diversity**: Measure vocabulary diversity and richness
- **Longest Word**: Identify the longest word in your text
- **Keyword Analysis**: Identify the most frequently used words with filtering options

### Text Processing Tools
- **Copy to Clipboard**: Quickly copy text to your clipboard
- **Download as .txt**: Export your text as a downloadable file
- **Trim Spaces**: Remove extra whitespace automatically
- **Case Conversion**: 
  - UPPERCASE: Convert to uppercase
  - lowercase: Convert to lowercase
  - Sentence Case: Apply proper sentence casing
- **Clear**: Clear the text area quickly with one click

### Smart Filtering
- **Ignore Common Words**: Toggle to exclude common words (a, the, and, etc.) from keyword analysis
- Live status messages to guide your usage

## Technology Stack

- **HTML5**: Semantic markup and modern structure
- **CSS3**: Modern styling with gradient backgrounds, blur effects, and responsive design
- **Vanilla JavaScript**: Fast, efficient text processing without dependencies

## Design Features

- Clean, professional interface with modern typography
- Responsive layout that works on desktop and tablet
- Smooth animations and transitions
- Real-time analysis as you type
- Hero section displaying session snapshot

## How to Use

1. Open `index.html` in your web browser
2. Start typing or paste your text in the text area
3. View live statistics on the right side
4. Use toolbar buttons to manipulate your text
5. Export or download your work as needed

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge) with ES6 support.

## Installation

No installation required! Simply download the files and open `index.html` in your browser.

## Tips for Best Results

- Use the "Load sample" button to see the tool in action
- Toggle "Ignore common words" to focus on meaningful keywords
- Set reading time estimates based on your target audience
- Use the trim function to clean up pasted text automatically
